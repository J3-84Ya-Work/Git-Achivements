# GitHub Achievements Guide

## YOLO
Merge a pull request without code review. Living on the edge!

## Quickdraw
Close an issue or pull request within 5 minutes of opening it.

## Pull Shark
Get pull requests merged. Tiers: Bronze (2), Silver (16), Gold (128).

## Pair Extraordinaire
Co-author commits on a merged pull request.

## Galaxy Brain
Get answers accepted on GitHub Discussions. Tiers: Bronze (2), Silver (8), Gold (16).

## Starstruck
Have a repository reach 16+ stars. Tiers: Bronze (16), Silver (128), Gold (512).
